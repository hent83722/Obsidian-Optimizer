# Contributing to Obsidian Optimizer

Thank you for wanting to contribute! 🚀  
This project is open source, and contributions are always welcome.

---

## 🐛 Reporting Bugs
- Check if the issue already exists in the [Issues tab](../../issues).
- If not, open a **new Issue** and include:
  - What happened
  - Steps to reproduce
  - What you expected to happen
  - Server version (Minecraft + Paper/Spigot version)
  - Plugin version

---
